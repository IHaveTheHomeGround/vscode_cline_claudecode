```

```

> 또는 대화 중 "업무 모드" / "취미 모드" 로 전환 요청 가능.

---

## 환경-프로파일 매핑 (이 사용자 기준)

| 장소 | OS | 프로파일 | 비고 |
|------|----|----------|------|
| 회사 | Windows 10 | **WORK** | 백업 필수, 로그 필수, 스크립트: `bash backup.sh` (Git Bash) |
| 집 | Mac | **HOBBY** | 백업 선택, 로그 선택, 스크립트: `bash backup.sh` (Terminal) |

> **Mac Terminal과 회사 Git Bash 모두 동일한 `.sh` 스크립트를 사용합니다.**  
> Windows PowerShell / `.bat` 파일은 사용하지 않습니다.

---

## WORK 프로파일

**적용**: 회사 업무, 클라이언트 프로젝트

| 항목 | 설정 |
|------|------|
| 백업 정책 | STANDARD 이상 필수 |
| 로그 기록 | 모든 작업 필수 |
| 승인 임계값 | CRITICAL만 |
| 보고 스타일 | 구조화된 형식 + 기술 상세 |
| 체크포인트 보존 | 90일 |
| 드라이런 | 스크립트 항상 필수 |

**특이사항**: 작업 미완료 상태로 세션 종료 금지. 중단 시 반드시 현재 상태 보고.

---

## HOBBY 프로파일

**적용**: 개인 프로젝트, 학습, 실험

| 항목 | 설정 |
|------|------|
| 백업 정책 | CRITICAL만 필수 |
| 로그 기록 | 선택사항 |
| 승인 임계값 | CRITICAL만 |
| 보고 스타일 | 간결하게 |
| 체크포인트 보존 | 30일 |
| 드라이런 | 권장하지만 선택 |

**특이사항**: 실험적 시도 허용. 실패도 학습 자료로 기록.

---

## 프로파일 전환 보고 형식

```
[프로파일 전환]
- WORK → HOBBY
- 백업 정책: STANDARD 이상 필수 → CRITICAL만 필수
- 로그: 필수 → 선택
```
